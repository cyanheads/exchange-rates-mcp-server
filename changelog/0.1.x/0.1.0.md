# 0.1.0 — 2026-06-04

## Added

- **`fx_list_currencies`** — list all ~30 ECB-supported ISO 4217 currencies with full names.
- **`fx_get_rates`** — bulk snapshot of all rates for a base currency; optional `symbols` filter.
- **`fx_get_rate`** — exchange rate for a currency pair on a given date; cross-rates triangulated through EUR automatically; `date_snapped` flag when weekend/holiday request returns prior business-day rate.
- **`fx_convert_currency`** — convert an amount between any two currencies at latest or historical rate; surfaces `date_snapped` and full provenance fields.
- **`fx_get_timeseries`** — historical daily rates over a date range; short ranges (≤90 days, configurable) returned inline; long ranges spilled to DataCanvas with `canvas_id` + `table_name` for SQL follow-up.
- **`fx_dataframe_describe`** — list tables and columns staged on a DataCanvas from `fx_get_timeseries`.
- **`fx_dataframe_query`** — run read-only SQL SELECT against DataCanvas tables; supports aggregations, GROUP BY, window functions, JOINs.
- **`fx://currencies`** resource — stable reference document for all supported currencies.
- **`fx://rates/latest/{base}`** resource — latest rates snapshot for any base currency as a stable URI.
- **`FrankfurterService`** — wraps `api.frankfurter.dev/v1`; keyless, Cloudflare-fronted ECB proxy; `withRetry` on all HTTP calls; clear 404→unsupported-currency error mapping.
- **DataCanvas integration** — `@duckdb/node-api` peer dep; `CANVAS_PROVIDER_TYPE=duckdb` enables spillover for long time-series; `canvas-accessor.ts` module-level accessor.
- **Server config** — `FRANKFURTER_BASE_URL` (override API endpoint) and `FX_TIMESERIES_CANVAS_THRESHOLD_DAYS` (spillover threshold, default 90).
- All tools carry `rate_type: "ECB reference (mid-market)"` and `source: "ECB via Frankfurter"` provenance fields in every response.
