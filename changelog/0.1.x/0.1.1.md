# 0.1.1 ‚Äî 2026-06-05

Initial public release of `@cyanheads/exchange-rates-mcp-server`. Provides 7 MCP tools and 2 resources over the [Frankfurter API](https://www.frankfurter.dev/) (ECB reference foreign-exchange rates):

- **`fx_list_currencies`** ‚Äî list all ~30 ECB-supported ISO 4217 currencies with full names
- **`fx_get_rates`** ‚Äî snapshot of all rates for a base currency at latest or a historical date
- **`fx_get_rate`** ‚Äî exchange rate for a single currency pair at latest or a historical date; surfaces `date_snapped` when a weekend/holiday request returns the prior business-day rate
- **`fx_convert_currency`** ‚Äî convert an amount between any two currencies; cross-rates triangulated through EUR
- **`fx_get_timeseries`** ‚Äî historical daily rates for a currency pair over a date range; ranges ‚â§90 days returned inline, longer ranges spill to DataCanvas with a `canvas_id` for SQL follow-up
- **`fx_dataframe_describe`** ‚Äî list DataCanvas tables and columns from a prior `fx_get_timeseries` call
- **`fx_dataframe_query`** ‚Äî read-only SQL SELECT against DataCanvas tables; supports aggregations, GROUP BY, window functions, and JOINs

Resources: **`fx://rates/latest/{base}`** (latest rates snapshot) and **`fx://currencies`** (stable currencies reference document).

## Added

- **`fx_list_currencies`**, **`fx_get_rates`**, **`fx_get_rate`**, **`fx_convert_currency`**, **`fx_get_timeseries`**, **`fx_dataframe_describe`**, **`fx_dataframe_query`** ‚Äî full 7-tool surface over the Frankfurter API
- **`fx://rates/latest/{base}`** resource ‚Äî stable URI for latest rates snapshot per base currency
- **`fx://currencies`** resource ‚Äî stable currencies reference document
- **DataCanvas support** (`CANVAS_PROVIDER_TYPE=duckdb`) ‚Äî `fx_get_timeseries` spills long date ranges to named DuckDB tables for follow-up SQL
- **`Dockerfile`** ‚Äî copies `node_modules` from the build stage so `@duckdb/node-api`'s pre-compiled native binary is preserved in the production image without reinstalling build tools

## Security

- **Input injection hardening in `frankfurter-service.ts`** ‚Äî added `assertCurrencyFormat()` (regex `/^[A-Za-z]{1,4}$/`) and `assertDateFormat()` (regex `/^\d{4}-\d{2}-\d{2}$/`) guards applied to all `base`, `quote`, `symbols`, `startDate`, and `endDate` inputs before they are embedded in API URLs
- **URL parameter encoding** ‚Äî switched from string interpolation to `URLSearchParams` and `encodeURIComponent()` across all `FrankfurterService` methods
